Detailed Description

Class for representing PIDRegulator that controls temperature according to e(t) - deviation of the output variable y(t) from the desired value w(t).

double t )
inlineoverridevirtual

Calculate next u - input warm according to e & t - and returns it.

Formula:

\[u(k) = u(k - 1) + \Delta u(k) \]

\[\Delta u(k) = q_1 e(k) + q_2 e(k - 1) + q_3 e(k - 2) \]

\[q_0 = K \left( 1 + \frac{T_D}{T_0} \right) \]

\[q_1 = -K \left( 1 + 2 \frac{T_D}{T_0} - \frac{T_0}{T} \right) \]

\[q_2 = K \frac{T_D}{T_0} \]
